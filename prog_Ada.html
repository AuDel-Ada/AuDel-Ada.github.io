ADA TECH SCHOOL : Formation en 08 mois

Que se passe-t-il pendant les 08 mois précédent l'alternance ?

Juin 2021 - Janvier 2022

L'alternance est déjà de rigueur dans les locaux d'ADA : pendant les premiers mois de formation, les apprenant.e.s enchaînent apprentissage de notions et projets collectifs.
Tout au long de ce parcours : pratiques, échanges, code reviews, pair- (voire mob-) programming sont de rigueur, poour allier développement des techniques et des soft kills.

  • Juin
        • Variables, flux d'éexécution et structures de contrôle
        Fonctions et bibliothèques
  • Juillet
        • Projet collectif : créer un jeu rétro avec Pico-8
        Structures de données
        Programmation asynchrone
        Projet collectif : organiser une visualisation de données
  • Août
        • Base de développement web
        Clean code et refactorisation
  • Septembre
        • Projet collectif : développer une extension de navigateur
        Programmation orientée objet
        Base de données relationnelles
  • Octobre
        • Projet collectif : créer un réseau social en PHP
        Tests et TDD
        Framework front-end
  • Novembre
        • Projet collectif : back + front, création d'une plateforme de vente
        Choix architecturaux et patrons de conception
        Projet collectif : développement d'une plateforme de micro-blogging (Laravel)

  • Janvier
